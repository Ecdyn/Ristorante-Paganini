## Spacing Scale

All tokens defined in `:root` of `assets/css/main.css`. Executor MUST use CSS custom property names — never hard-code px values.

| Token | CSS Variable | Value | Usage in Phase 4 |
|-------|-------------|-------|------------------|
| xs | `--space-1` | 4px (0.25rem) | Icon gaps, decorative rule margin |
| sm | `--space-2` | 8px (0.5rem) | Compact element padding |
| md | `--space-4` | 16px (1rem) | Default element spacing, flow spacing |
| lg | `--space-6` | 24px (1.5rem) | Wrapper padding, cluster gaps |

## Color

All tokens defined in `:root` of `assets/css/main.css`. Executor MUST reference CSS variables — never hard-code hex values.

| Role | CSS Variable | Hex Value | Usage |
|------|-------------|-----------|-------|
| Dominant surface (60%) | `--color-surface` | #FBF5F5 | Page background, header (scrolled), cards on light sections |
| Secondary surface (30%) | `--color-surface-alt` | #F5EDEE | Alternate section backgrounds (`.section--alt`) |
| Dark surface | `--color-surface-dark` | #2C1A1D | Footer (`.section--dark`), mobile menu overlay |
| Accent (10%) | `--color-accent` | #7A2D3A | Reserved for specific elements only — see list below |
| Accent hover | `--color-accent-hover` | #5E2230 | Accent element hover/focus state only |
| Accent light | `--color-accent-light` | #9E4455 | Links/hover on dark backgrounds |
| Text primary | `--color-text` | #2A1F20 | All body text on light surfaces |
| Text muted | `--color-text-muted` | #6B5557 | Supporting copy, captions, meta info |
| Text inverse | `--color-text-inverse` | #FBF5F5 | Text on dark surfaces (footer, mobile menu, hero overlay) |
| Border | `--color-border` | #DFD0D2 | Dividers, card borders, input borders |
| Hero overlay | `--color-overlay` | rgba(28, 12, 14, 0.55) | Hero section dark overlay only |

Accent (`--color-accent` #7A2D3A) is reserved for:
1. Primary CTA button fill and border (`data-variant="primary"`)
2. Secondary CTA button border and hover fill (`data-variant="secondary"`)
3. H2 decorative underline rule (`:after` pseudo-element)
4. Navigation link active/hover state (`.nav a:hover`, `.nav a[aria-current]`)
5. Default link color (`a { color: var(--color-accent) }`)
6. Focus ring outline (`:focus-visible`)
7. Gallery caption background on hover (new — see Interaction Contracts)
8. Business lunch highlight slot accent border (new — see Section Contracts)

Accent is NOT used for: section backgrounds, body text, muted text, borders, or decorative elements not in the list above.

Destructive: No destructive actions in this phase. No destructive color token needed.

Source: `assets/css/main.css` lines 42–52, `01-CONTEXT.md` D-01–D-04.

## Interaction Contracts

### Gallery Lightbox

- Trigger: click on any gallery image
- Behavior: `.lightbox` modal appears with `opacity` transition (`--transition-base` 250ms)
- Controls: previous (`←`) and next (`→`) buttons, close button (top-right, "×")
- Keyboard: Escape closes, ArrowLeft/ArrowRight navigate
- ARIA: `role="dialog"`, `aria-modal="true"`, `aria-label="Bildergalerie"`
- Focus trap: focus cycles within lightbox when open; returns to triggering image on close
- Mobile: navigation via tap on left/right halves of image (no swipe — deferred per CONTEXT.md)
- `prefers-reduced-motion`: skip opacity transition, show immediately
- JS added to `assets/js/main.js`

### Gallery Hover Captions

- Trigger: `:hover` on `.gallery__item` (desktop), `:focus-visible` on image link (all)
- Caption container: absolute positioned, `inset: 0`, `background: rgba(28, 12, 14, 0.8)`
- Caption text: Lato `--font-size-sm`, `--color-text-inverse`, centered vertically
- Transition: `opacity var(--transition-base)`, default opacity 0, hover opacity 1
- Touch devices: caption visible on tap (toggle via JS), or omit on touch — executor decides
- `prefers-reduced-motion`: show caption without transition (instant)

### FAQ Accordion (if `<details>` approach chosen)

- `<details>` open/close is browser-native — no JS required
- Optional: CSS animation on `<details>[open]` content via `animation: slideDown var(--transition-base)`
- `prefers-reduced-motion`: no animation

### Reservation Phone Fallback

- On mobile: "Rufen Sie uns an" button links via `href="tel:..."` — triggers native phone dialer
- No JS interaction needed

### Navigation Anchor Scrolling

- Already implemented in Phase 2 — `scroll-behavior: smooth` on `html` element, `scroll-padding-top: var(--header-height)` (72px)
- Mobile menu closes on link click — already implemented in `assets/js/main.js`
