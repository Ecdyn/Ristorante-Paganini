Create `index.html` at the project root (NOT inside any subdirectory). The file is a self-contained redirect utility with zero external stylesheet or script dependencies. The complete structure:

**Head section (in this exact order):**
1. `<!DOCTYPE html>` and `<html lang="de">` (German is the default/primary language)
2. `<meta charset="UTF-8">` (must be first meta)
3. `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
4. `<title>Ristorante Paganini</title>` (simple title, no SEO needed for noindex page)
5. `<meta name="robots" content="noindex, nofollow">` (per D-05/D-12 — keeps page out of search indexes)
6. `<noscript><meta http-equiv="refresh" content="0;url=/de/"></noscript>` (per D-06 — must be inside `<head>`, not `<body>`, for meta refresh to work correctly in all browsers)
7. Favicon links using `assets/` prefix (NOT `../assets/`):
   - `<link rel="icon" href="assets/favicon.svg" type="image/svg+xml">`
   - `<link rel="icon" href="assets/favicon.ico" sizes="32x32">`
   - `<link rel="apple-touch-icon" href="assets/apple-touch-icon.png">`
   - `<link rel="manifest" href="assets/site.webmanifest">`
8. Inline `<style>` block (no external CSS link — per research, avoids blocking HTTP request on a page the user exits in under 100ms):

```css
*, *::before, *::after { box-sizing: border-box; margin: 0; padding: 0; }
html, body {
  height: 100%;
  background-color: #2C1A1D;
  display: flex;
  align-items: center;
  justify-content: center;
}
.redirect-flash {
  color: #FBF5F5;
  font-family: 'Cormorant Garamond', Georgia, serif;
  font-size: clamp(1.5rem, 4vw, 2.5rem);
  font-weight: 400;
  letter-spacing: 0.06em;
  text-align: center;
  opacity: 0.9;
}
```

**Body section:**
1. Branded flash element: `<div class="redirect-flash" aria-hidden="true">Ristorante Paganini</div>` (per D-01/D-10 — aria-hidden because it is decorative, not content)
2. Inline `<script>` block at end of body with language detection IIFE (per D-02/D-03/D-07/D-08/D-09):

```javascript
(function () {
  'use strict';
  var supported = { de: '/de/', en: '/en/', it: '/it/' };
  var langs = navigator.languages && navigator.languages.length
    ? navigator.languages
    : [navigator.language || 'de'];

  for (var i = 0; i < langs.length; i++) {
    var prefix = langs[i].toLowerCase().split('-')[0];
    if (supported[prefix]) {
      window.location.replace(supported[prefix]);
      return;
    }
  }
  window.location.replace('/de/');
}());
```

Key implementation rules:
- Do NOT link `../assets/css/main.css` or any external stylesheet (per research — blocks rendering for a zero-content page)
- Do NOT add `@font-face` declarations — Georgia serif fallback is sufficient for the ~100ms flash
- Do NOT add hreflang tags — this page is not part of the hreflang cluster (it is noindex)
- Do NOT add canonical link — redirect-only pages should not claim canonical status
- Do NOT use `window.location.href =` — must be `window.location.replace()` to prevent back-button loops (per D-02)
- Do NOT use cookies or localStorage for language persistence (per D-03)
- Do NOT add any visible content beyond the branded flash div (per D-04/D-10)
- Use root-relative paths (`/de/`, `/en/`, `/it/`) in all redirect targets — not relative (`./de/`)
- Add an HTML comment near the top: `<!-- Root redirect page — not indexed, not in sitemap. See Phase 8. -->`
- Add an HTML comment before the script: `<!-- Language detection: iterates navigator.languages, redirects to first supported match, defaults to /de/ -->`
